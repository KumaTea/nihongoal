# NihonGoal!

An AI-native Japanese-learning companion that helps people learn through their
own daily life—not only through isolated vocabulary drills.

**NihonGoal!** combines *Nihongo* (Japanese) with a personal learning goal:
one useful, achievable step at a time.

## Project status

An Expo web prototype is implemented and is designed to remain portable to iOS
and Android. It includes an adaptive Sensei, user-configured AI connections,
text/image discovery, source-linked news reading, and local learning data.

## Working direction

- Begin with a web prototype to validate the experience quickly.
- Keep product concepts and core logic portable so a future mobile app is a
  first-class destination, not an afterthought.
- Make AI assistance the central product capability.
- Support interchangeable AI providers/models; a user may choose their own
  compatible provider and credentials.
